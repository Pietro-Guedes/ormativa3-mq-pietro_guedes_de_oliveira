💻 Formativa 
Este projeto foi desenvolvido para atender clientes online da pizzaria BellaPizza.

🛠️ Tecnologias Utilizadas
HTML5 (Semântica e Acessibilidade)
CSS3 (Estilização e Feedback Visual)
JavaScript (Manipulação de DOM e Eventos)
📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado: / ├── index.html └── src/ ├── styles/ (Estilos da página) ├── scripts/ (Lógica de interação) └── images/ (Ativos visuais)

🚀 Funcionalidades Implementadas
Interatividade Visual: Usei o media query e defini quando a largura da tela ficasse em 768, o que tinha 3 colunas, muda para 2, e e depois foi feiot outro media que pra baixo de 480px, fica uma coluna, o titulo fica na vertical usando flex direction column e aling intems pare centralizar ele e a dica que aparece antes, deseparece usando o display: none.
